# Code of Conduct

## Our pledge

We want Universal Context Layer to be a welcoming, respectful project for everyone.

Participants are expected to:

- be respectful and constructive
- assume good intent while giving direct, useful feedback
- focus on the ideas and the implementation, not the person
- help keep the project accessible to contributors with different backgrounds and experience levels

## Unacceptable behavior

Examples of unacceptable behavior include:

- harassment, discrimination, or personal attacks
- trolling, intimidation, or deliberately inflammatory behavior
- publishing private information without permission
- sustained disruption of discussions, issues, or reviews

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or other contributions that do not align with this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately through GitHub to the repository maintainer.

## Scope

This Code of Conduct applies within project spaces and in public spaces when an individual is representing the project.
