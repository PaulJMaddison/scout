# KynticAI Scout v2.5.1

KynticAI Scout v2.5.1 was a company website hosting activation patch for the earlier brochure/demo site introduced in v2.5.0. That temporary website path was retired from the open-source repo in v2.8.0.

## Highlights

- Enabled company website hosting for the repository.
- Published the preview demo to the Pages branch.
- Added the live preview URL to the README and preview documentation.
- Set the repository homepage URL to the live Pages site.
- Added `.nojekyll` generation to the preview build output.

## Verification

- `npm run build:web app`
- `npm run lint`
- company website hosting API reports `built`
- Live URL returns HTTP 200
- Rendered Playwright check confirms the React bundle loads, including Northstar Logistics, full downloadable product clarification, and FAQ content
