# Universal Context Layer v1.1

Universal Context Layer v1.1 is the messaging and demo-readiness release. It keeps the v1.0 public demo foundation and sharpens the first impression so every page answers the buyer question: what does this do for me?

## Highlights

- Reworked the landing hero to say the product outcome plainly: turn existing business data into trusted context.
- Made the main product pages outcome-led, including dashboard, data sources, selector builder, schema registry, customer context viewer, example sales support consumer, audit log, Bootstrap Studio, and the guided walkthrough.
- Expanded README instructions for a fresh laptop install using the default SQLite path, including repo-local .NET and Node bootstrap, start, reset, and verification commands.
- Refreshed README screenshots from the latest running SQLite-backed UI.
- Removed GitHub Actions workflows so the public repo does not consume paid Actions minutes.

## Why It Matters

The product now reads better for strangers, CEOs, CTOs, and technical evaluators. Instead of leading with internal feature labels, the UI explains the commercial or operational value of each screen before asking the user to inspect tables, mappings, schemas, or audit records.

## Verification

- `npm.cmd run lint --prefix apps/web`
- `npm.cmd run build --prefix apps/web`

The underlying v1.0 test suite remains documented in the README for local verification.
