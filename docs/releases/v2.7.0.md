# KynticAI Scout v2.7.0

KynticAI Scout v2.7.0 prepares the public open-core product for live hosting and paid-pilot readiness while keeping the customer data-plane boundary clear.

## Highlights

- Customer data-plane installation readiness, including Windows/Linux runbooks and production-style configuration checks.
- Public demo improvements that keep the product story clear without implying complete self-serve SaaS functionality.
- Paid-pilot documentation for rehearsal, support expectations, production readiness, hosting alignment, and customer handover.
- Public API, SDK, and context-package documentation for REST, GraphQL, TypeScript, .NET, machine-to-machine auth, and AI-safe context package usage.
- Legal, privacy, cookie/consent, terms, support, and operator checklist drafts for legal review before customer use.
- Observability and machine-to-machine identity guidance for production-style deployments.

## Boundaries

The public repository remains open core. It does not include paid enterprise connector implementations, private cloud control-plane code, customer-specific deployment packs, real licence signing keys, secrets, or raw customer operational data.

This release supports public demo, live-hosting preparation, and paid-pilot conversations. It is not a complete self-serve SaaS release.
