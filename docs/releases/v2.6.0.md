# Universal Context Layer v2.6.0

Universal Context Layer v2.6.0 strengthens the public open-core product story and integration surface around the customer-owned data plane.

## Highlights

- Clearer positioning: UCL does not build the brain; it builds the nervous system for trusted semantic context.
- Expanded customer data-plane, buyer FAQ, integration-layer, public API, paid-pilot, and static-demo documentation.
- Richer static GitHub Pages demo using fictional data only.
- REST, GraphQL, TypeScript SDK, and .NET SDK examples/tests for context lookup, semantic facts, snapshots, AI-safe context packages, selector preview/validation, events, auth, errors, filtering, and pagination.
- Public connector catalogue labels that distinguish open-core examples, paid enterprise implementations, planned connectors, and customer-specific work.

## Boundaries

The public repo remains open core. It does not include paid enterprise connector implementations, private cloud billing/control-plane code, real licence signing secrets, customer-specific implementation code, or raw customer data.

This release supports public demo and paid-pilot conversations. It is not a complete self-serve SaaS release.
