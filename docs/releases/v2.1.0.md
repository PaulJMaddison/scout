# KynticAI Scout v2.1.0

Release date: 2026-05-11

## Summary

v2.1.0 prepares KynticAI Scout for a public open-core relaunch after the private enterprise and cloud implementation streams. The public repository contains the safe open core and self-hosted data-plane foundations. Paid enterprise and cloud capabilities are marketed clearly but their implementation code remains in private repositories.

## Public Open-Core Scope

Included in the public repo:

- local SQLite demo with fictional data
- self-hosted data-plane foundations
- React marketing site, demo, and admin console
- REST and GraphQL APIs
- SDK scaffolds and examples
- API key and machine-client foundations
- connector plugin interfaces
- SQL, REST API, CSV, mock CRM, mock billing, and mock support connectors
- blueprint import from user-provided JSON
- governance, licence, usage, audit, and deployment foundations

Not included in the public repo:

- real Salesforce, HubSpot, Dynamics, Snowflake, BigQuery, Zendesk, or NetSuite connector implementations
- SSO/SAML or SCIM implementations
- Stripe, Paddle, or other live payment-provider code
- hosted account, billing, licence portal, download portal, update channel, support portal, or cloud operations code
- private signing keys, vendor credentials, customer-specific code, or real customer data

## Verification Intent

Release verification must confirm builds/tests, frontend lint/build, SDK tests, public boundary scans, secret scans, workflow absence, and GitHub visibility. Enterprise and cloud repositories must remain private.
