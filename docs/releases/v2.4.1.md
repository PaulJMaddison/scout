# KynticAI Scout v2.4.1

KynticAI Scout v2.4.1 is a paid-ad readiness patch release for the public open-core site and pilot capture path.

## Highlights

- Dedicated `/pilot` landing page with a single paid pilot CTA.
- Lead capture fields for name, work email, company, source systems, and target workflow.
- Configurable `VITE_PILOT_LEAD_ENDPOINT` so the public pilot form can submit to the private cloud mini CRM.
- Privacy and terms/disclaimer pages for a safer public launch baseline.
- Cleaner marketing headings without "This page..." public-facing copy.

## Commercial Boundary

This release keeps the public repo as open-core product, documentation, demo, and customer-owned data-plane positioning. It does not add paid vendor connector implementation code, customer operational data processing in the public site, or complete self-serve SaaS operations.
