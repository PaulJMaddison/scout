# KynticAI Scout v1.0

KynticAI Scout v1.0 is the first CEO and CTO ready local demo release. It shows how fragmented customer operations data can stay in place while a semantic data plane turns it into reusable, explainable AI context.

## Highlights

- Polished executive walkthrough covering legacy signals, semantic transformation, AI interactions, and rollout ROI.
- Zero-friction default laptop demo using two local SQLite database files, with repo-local .NET 10 and Node.js bootstrap for fresh machines.
- Optional Docker/PostgreSQL package mode for production-like demos and observability.
- Two database architecture with customer operations data separated from semantic context, selectors, provenance, AI prompts, and audit history.
- Bootstrap Studio flow for using Codex, Claude, or another AI engineering assistant to analyse existing CRM, warehouse, or product databases and generate an importable Scout blueprint.
- Selector builder, semantic schema registry, customer context viewer, example grounded sales support consumer, and audit/provenance screens.
- Seeded Larkspur Logistics Group demo record showing how `User 123` becomes a context rich customer profile with conversion probability, preferred channel, enterprise plan interest, churn risk, expansion potential, and recommended sales motion.
- Responsive UI hardening for laptop, desktop, and mobile viewports, including regression tests for the login page and admin routes.
- Refreshed README screenshots captured from the running app.
